This program takes a given string of letters and can encode/decode the string 
using a default algorithm, a caesar cipher alphabet, a cryptogram alphabet or a keyboard cipher alphabet.

Tested On:
ubuntu 18.04

*Note Python needs to be installed

To run execute in a command prompt
>python3 cryptogram.py

Testing:
This program can take user input or a text file as input.
